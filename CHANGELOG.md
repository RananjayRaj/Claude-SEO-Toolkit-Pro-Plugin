# Changelog

All notable changes to SEO Toolkit Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2025-06-17

### Added

- Technical SEO Auditor - full site health checks covering crawlability, indexation, redirects, mobile usability, and security
- Keyword Clustering + Intent Mapper - semantic grouping with intent classification and funnel stage mapping
- Topical Authority Planner - pillar/cluster content strategy with 6-month roadmap
- SEO Content Brief Generator - complete briefs with heading structure, semantics, competitor gaps, and FAQs
- SEO Keyword Optimization - on-page optimization for meta tags, title tags, and keyword placement
- AEO Answer Engine Optimizer - optimization for Google AI Overviews, Perplexity, Bing Copilot, and voice search
- GEO Content Optimizer - content structuring for citation by ChatGPT, Claude, and Gemini
- Content Cannibalization Detector - identifies pages competing for the same keywords with consolidation recommendations
- Core Web Vitals Optimizer - LCP, INP, and CLS diagnosis with specific code-level fixes
- Internal Linking Strategist - PageRank distribution, orphan page fixes, and anchor text strategy
- Local SEO Playbook Generator - GBP optimization, citation targets, review templates, and NAP audits
- Schema Tag Generator - JSON-LD structured data for Article, FAQ, Product, LocalBusiness, HowTo, Event, and more
- 12 slash commands mapping to each skill
- Natural language triggering for all skills
